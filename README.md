# AI Spam Shield

AI Spam Shield is a spam email detection system built using DistilBERT and Gradio.

## Features

* Spam Detection
* Ham Detection
* Confidence Score
* Example Messages
* Modern User Interface
* Responsive Design

## Technology Stack

* Python
* Gradio
* Transformers
* DistilBERT
* Hugging Face

## Installation

pip install -r requirements.txt

## Run Project

py app.py
